## 快速参考卡片

### 安装命令

```bash
# Claude Code
curl -fsSL https://claude.ai/install.sh | bash

# Codex CLI
npm install -g @openai/codex

# Copilot CLI（Public Preview，需 Node.js 22+）
npm install -g @github/copilot
```

### MCP 命令

```bash
claude mcp list              # 列出服务器
claude mcp add [name] ...    # 添加服务器
claude mcp remove [name]     # 移除服务器
/mcp                         # 查看连接状态
```

### Cursor 快捷键

| 功能 | macOS | Windows |
|:---|:---|:---|
| Chat | `⌘ + L` | `Ctrl + L` |
| Inline Edit | `⌘ + K` | `Ctrl + K` |
| Composer | `⌘ + I` | `Ctrl + I` |

### 提示词技巧

- 具体明确，避免模糊
- 分步骤，不要一次太多
- 提供上下文（技术栈、背景）
- 给示例，比描述更清晰
- 先原型、再优化、再正式

### 安全提醒

- 不在提示词中包含敏感信息
- 审查涉及安全的代码
- 原型可以随意，生产要谨慎
